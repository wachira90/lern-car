สำหรับรถยนต์ Honda Brio ที่ใช้เครื่องยนต์ 1.2 ลิตร (รหัส L12B) และเกียร์อัตโนมัติ CVT จะมีเซ็นเซอร์หลักๆ ที่ทำงานร่วมกันเพื่อประมวลผลให้รถวิ่งได้สมบูรณ์อยู่ประมาณ **12-14 ตัวหลัก** ครับ (จำนวนสุทธิอาจต่างกันเล็กน้อยขึ้นอยู่กับว่านับรวมระบบแวดล้อมอย่างเบรก ABS หรือระบบแอร์ด้วยหรือไม่)

เราสามารถแบ่งกลุ่มเซ็นเซอร์หลักๆ ตามหน้าที่การทำงานออกเป็น 2 ระบบใหญ่ๆ ได้ดังนี้ครับ:

## เซ็นเซอร์ระบบเครื่องยนต์ (Engine Sensors)

กลุ่มนี้ทำหน้าที่ส่งข้อมูลไปยังกล่อง ECU เพื่อคำนวณการจ่ายน้ำมันและจังหวะจุดระเบิดให้เครื่องยนต์ทำงานได้เต็มประสิทธิภาพ:

* **O2 Sensor (ออกซิเจนเซ็นเซอร์):** ใน Brio จะมี 2 ตัว (ตัวหน้าบริเวณท่อร่วมไอเสีย และตัวหลังแคตตาไลติก) ทำหน้าที่วัดปริมาณออกซิเจนในไอเสีย หากส่วนผสมหนาหรือบางไป ECU จะสั่งปรับการฉีดน้ำมันให้สมดุลที่สุด
* **Crankshaft Position Sensor (เซ็นเซอร์เพลาข้อเหวี่ยง):** วัดความเร็วรอบเครื่องยนต์และตำแหน่งของเพลาข้อเหวี่ยง เพื่อให้ ECU รู้ว่าตอนนี้เครื่องยนต์หมุนไปกี่องศาแล้ว และควรสั่งจ่ายน้ำมันจังหวะไหน
* **Camshaft Position Sensor (เซ็นเซอร์เพลาลูกเบี้ยว):** ทำงานคู่กับเซ็นเซอร์เพลาข้อเหวี่ยง เพื่อระบุตำแหน่งของวาล์วและลูกสูบ ช่วยให้ระบบจุดระเบิดได้อย่างแม่นยำในแต่ละสูบ
* **MAP Sensor (Manifold Absolute Pressure):** วัดแรงดันและสุญญากาศภายในท่อร่วมไอดี เพื่อประเมินว่าเครื่องยนต์กำลังรับภาระหนักแค่ไหน (เช่น ตอนเร่งแซง หรือตอนเดินเบา)
* **IAT Sensor (Intake Air Temperature):** วัดอุณหภูมิของอากาศที่ดูดเข้าเครื่องยนต์ (มักติดอยู่ใกล้หรือรวมกับ MAP Sensor) อากาศเย็นจะมีความหนาแน่นมากกว่าอากาศร้อน ECU จึงต้องใช้ค่านี้คำนวณปริมาณน้ำมันให้สอดคล้องกัน
* **ECT Sensor (Engine Coolant Temperature):** วัดอุณหภูมิน้ำหล่อเย็น หากเครื่องเย็นระบบจะสั่งจ่ายน้ำมันหนาขึ้นเพื่อให้สตาร์ทติดง่าย และทำหน้าที่สั่งพัดลมหม้อน้ำให้ทำงานเมื่อถึงอุณหภูมิที่กำหนด
* **TPS (Throttle Position Sensor) และ APP (Accelerator Pedal Position):** ตรวจจับองศาการเปิดของลิ้นปีกผีเสื้อและตำแหน่งการเหยียบคันเร่งไฟฟ้า เพื่อประเมินความต้องการของผู้ขับขี่และเร่งเครื่องตามน้ำหนักเท้า
* **Knock Sensor (น็อคเซ็นเซอร์):** ติดตั้งอยู่ข้างเสื้อสูบ ทำหน้าที่เหมือนหูฟังคอยจับเสียงสะเทือนที่เกิดจากการชิงจุดระเบิด (เครื่องเขก) หากพบความผิดปกติ ECU จะสั่งลดองศาไฟจุดระเบิดลงเพื่อป้องกันเครื่องยนต์เสียหาย

## เซ็นเซอร์ระบบเกียร์ CVT (Transmission Sensors)

ระบบเกียร์ CVT ของ Honda จะใช้เซ็นเซอร์เหล่านี้ส่งข้อมูลไปที่กล่องสมองกลเกียร์ (TCM) เพื่อปรับพูลเลย์และสายพานเหล็กให้ได้อัตราทดที่นุ่มนวลและเหมาะสมกับความเร็ว:

* **Primary / Input Pulley Speed Sensor:** เซ็นเซอร์วัดความเร็วรอบของ "พูลเลย์ตัวขับ" (ฝั่งที่รับแรงบิดมาจากเครื่องยนต์)
* **Secondary / Output Pulley Speed Sensor:** เซ็นเซอร์วัดความเร็วรอบของ "พูลเลย์ตัวตาม" (ฝั่งที่ส่งกำลังลงสู่เพลาขับ) กล่องเกียร์จะนำความเร็วของเซ็นเซอร์ตัวนี้และตัวบนมาเทียบกัน เพื่อคำนวณและกางพูลเลย์ปรับอัตราทดเกียร์ CVT แบบเรียลไทม์
* **TFT Sensor (Transmission Fluid Temperature):** วัดอุณหภูมิน้ำมันเกียร์ CVT หากขับขี่รุนแรงจนน้ำมันเกียร์ร้อนจัดเกินเกณฑ์ ระบบจะสั่งเข้าสู่ Safe Mode (ตัดกำลังเครื่อง หรือวิ่งได้ช้าลง) เพื่อป้องกันสายพาน CVT รูดและเกียร์พัง
* **Transmission Range Switch (เซ็นเซอร์ตำแหน่งเกียร์):** เป็นสวิตช์ตรวจจับว่าตอนนี้ผู้ขับขี่เลื่อนคันเกียร์ไปที่ตำแหน่งใด (P, R, N, D, S) เพื่อให้สมองกลสั่งงานระบบขับเคลื่อนได้ถูก และเป็นตัวป้องกันไม่ให้สตาร์ทรถติดหากไม่ได้อยู่ในเกียร์ P หรือ N
* **VSS (Vehicle Speed Sensor):** ตัวจับความเร็วของรถยนต์ ซึ่งในรถรุ่นใหม่มักจะนำสัญญาณมาจากเซ็นเซอร์ ABS ที่ล้อแต่ละข้าง เพื่อมาคำนวณอัตราทดและแสดงผลความเร็วบนหน้าปัดหน้าจอนั่นเอง
